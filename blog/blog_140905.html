Alt-Text | 14.09.05

I wanted to come up with a way to show alternative text on certain words or phrases when you hovered over it. Why? Well, I'm a funny guyI'm not, so I thought I could show off my wit, There are actually many uses for this feature not just to crack the occasional joke. I can show quotes, additional information to the reader, define acronyms or put explanations about certain bits of code. to those who are interested.

So, here's how I did it... I googled, 'jQuery hover dialog appear', and found this great how-to by Alessio Atzeni. This tutorial provided a starting point for me but I was looking for something slightly different that also fit my minimal solarized theme. I also wanted to treat short entries different from longer ones in how they appear. For example, in the first paragraph, hover the cursor over 'I'm a funny guy' and see a two word quip then over 'show off my with' and see a longer explanation. Depending on the window size you are viewing this page with they will show up differently.

I'll illustrate this using one of my favourite quotes from one of my favourite books,

"This planet has - or rather had - a problem, which was this: most of the people living on it were unhappy for pretty much of the time. Many solutions were suggested for this problem, but most of these were largely concerned with the movement of small green pieces of paper, which was odd because on the whole it wasn't the small green pieces of paper that were unhappy."
     - Douglas Adams, The Hitchhiker's Guide to the Galaxy

First, define the content in HTML (Don't worry about the citation for now) 

<p> This planet has - or rather had - a problem, which was this: most of the people living on it were unhappy for pretty much of the time. Many solutions were suggested for this problem, but most of these were largely concerned with the movement of small <a class="masterQuote" quoteID="q1">green pieces of paper</a><a class="quote" id="q1">Pretty sure this is about money, or Pokemon cards... not sure</a>, which was odd because on the whole it wasn't the small green pieces of paper that were unhappy.<p>
Most of that is just the quote except for the two <a> tags. The first tag, 
<a class="masterQuote" quoteID="q1">green pieces of paper</a>
is just the part of the quote we want to highlight for rollover. The second tag, 
<a class="quote" id="q1">Pretty sure this is about money, or Pokemon cards... not sure</a>
is the alt-text comment you want to make on rollover. Notice the quoteID in the first tag is the same as the id in the second tag? This is how we target the rollover text. Ok, next is the style via css, 
.masterQuote {
  cursor: default;
  color: #cb4b16;
}
.quote {
  display: none;
  position: absolute;
  width: auto;
  opacity: 0.95;
  border-radius: 10px;
  padding: 5px;
  margin: 5px 0px 5px 0px;
  cursor: pointer;
}
which you can change to your heart's desire, but make sure to keep display: none and position: absolute, everything else is just style. Finally, the JavaScript/jQuery to make everything go (note, you'll need to load jQuery!) 
$(document).ready(function () {
  $('.masterQuote').hover(function(event){
    var quoteID = $(this).attr('quoteID');
    var boxWidth = $('.container');
    var quoteWidth = $('.quote#'+quoteID)
    if (boxWidth.width() < quoteWidth.width()) {
      $('.quote#'+quoteID).css('width', boxWidth.width());
    } else {
      $('.quote#'+quoteID).css('left', event.pageX);
      $('.quote#'+quoteID).css('max-width', '400px');
    };
    $('.quote#'+quoteID).css('display', 'block');
  }, function() {
    $('.quote').css('display', 'none');
    $('.quote').css('width', 'auto');
    $('.quote').css('left', '');
    $('.quote').css('max-width', '');
  });
});
Ok, don't panicOk, panic a little then shape up. we can break this down line by line. The first and last line tell jQuery to run the script when the page is ready. The second line targets the masterQuote class from the first <a> tag and says on hover run something, and off hover run something else. The on hover something simply displays the alt-text, and the off hover something removes the slt-text. This is done by manipulating the display css style from none (off) to block (on). On roll-off display is set back to none.

Now to make sure we are showing the correct alt-text we need to use the target id's, remember q1? Do this by var quoteID = $(this).attr('quoteID'); which takes the variable stored in quoteID for the current (mouse on) link. The if () {...} else {...} is there to help out with alt-text formatting. Personal preference for me, you may not need it. It's basically saying, if the quote is shortI raised my hand into the thin air and grabbed 400 pixels then display the alt text in place, below the cursor. However, if the quote is long make is span the page below the link. Here's the final product,
