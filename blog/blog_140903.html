New Site | 14.09.03

Now that I've defended my thesis [yay!] I thought I was due for an update for my profesisonal website. The old site is a bit of an eyesore and doesn't feature a whole lot [read:any] of my work. Revamping my site has also given me an opportunity to learn more about web technologies and how HTML/CSS/JavaScript all function to produce modern websites.

Instead of starting from scratch I adopted the bootstrap framework. Bootstrap is open source, actively developed and simple to use once you get the hang of how HTML/CSS/JS interact with each other. Bootstrap is also built to be responsive meaning that whatever device you open the site with it adjusts to fit perfectly - try resizing this window to a very narrow width like on a phone... you see how the content and menu respond.

The next step involves defining your own style, colour scheme and layout. I chose to stick with a fairly clean layout, keeping only a sticky dropdown navbar up top. Thin, sans-serif font seems to be all the rage these days so I adopted "Helvetica Neue" which defaults to the browser sans-serif writing if that font is missing. For colour schemes there was naturally only one choice, solarized. I work mostly in the terminal and in text editors like vim and solarized is really the best options for people who look at text and code all day. The scheme is brilliant, a light or dark mode where background and font colours are complimentary and easily swapped with accent colours working well with either. It's high enough contrast without causing you to squint and not so contrasty that you get a headache. I personally use the dark scheme for my day-to-day but you can use either on this site by clicking the buttons in the footer. They look like this:

Solarized Dark Solarized Light

Lastly, you can write custom javascript and have it run in addition to the Bootstrap javascript. I wrote a jQuery function which changes the website style when you click on those solarized dark/light buttons above. I also wrote a javascript function which selects a dark or light theme based on the time of day you look at this site:

function myTime() {
  var currentDate = new Date();
  var currentHour = currentDate.getHours();
  if ((currentHour >= 6) && (currentHour < 18)) {
    $('.btn-light').trigger('click');
  };
};

This piece of code runs the light theme from 6 am to 5:59:59 pm and defaults back to the dark theme outside of those hours.

Lastly, I'm still in the process of adding all of my past research and code to my site and GitHub account so stay tuned. Also, if you like this website you can clone the whole thing off GitHub.
